# OP Variables

Exposed as `real` operating-point (OP) variables for post-processing (`.probe`/`.print` or waveform viewer): 

| Parameter | Description | Unit |
|---|---|---|
| i_cfe | FE capacitive current | A/m² |
| i_cde | DE capacitive current | A/m² |
| i_cint | INT capacitive current | A/m² |
| i_p | Polarization current | A/m² |
| i_leakint | INT leakage current | A/m² |
| i_tat_fe | FE cap current | A/m² |
| i_tat_int | FE cap current | A/m² |
| p | Polarization state (0-1) | 1 |
| pol | Polarization state | C/m² |
| c_depl_u | Up domain depletion capacitance | F/m² |
| c_depl_d | Down domain depletion capacitance | F/m² |
| c_depl_tot | Total depletion capacitance | F/m² |
| v_fecap | FE layer voltage | V |
| v_int | INT layer voltage | V |
| v_depl | DEPL layer voltage | V |
| v_leak | FE leak branch voltage | V |
